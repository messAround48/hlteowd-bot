# HLTEOWD — How Long 'Till End Of Work Day

Telegram-бот, который показывает, сколько осталось до конца рабочего дня и визуализирует прогресс.

## Установка

1. Получите токен у [@BotFather](https://t.me/BotFather) в Telegram

2. Создайте файл `.env` на основе примера:
```bash
cp .env.example .env
```

3. Укажите токен в файле `.env`:
```env
TG_BOT_TOKEN=ваш_токен_от_BotFather
```

4. Запустите бота:
```bash
docker compose up -d
```

> **Примечание:** образ берётся из GitHub Container Registry (`ghcr.io`).
> Убедитесь, что он опубликован, или используйте `docker compose pull` перед запуском.

### Просмотр логов
```bash
docker compose logs -f
```

### Остановка
```bash
docker compose down
```

## Настройка

В файле `main.py` в словаре `WORK_SCHEDULE` можно изменить расписание:
- Ключи: `0` — понедельник, ..., `6` — воскресенье
- Значения: кортеж `(начало, конец)` или `None` для выходного

Текущее расписание:
- Пн–Чт: 8:00 – 17:00
- Пт: 8:00 – 16:00
- Сб, Вс: выходной

## Использование

Отправьте `/start` боту и нажмите кнопку «Прогресс рабочего дня».
